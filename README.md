# Teslo API

1. Clone the project
2. Run the command: 

```
yarn install
```
3. Clone the file ```.env.template``` and rename it to ```.env```
4. Change the environment variables
5. Run the database

```
docker-compose up -d
```

6. Run the command: 

```
yarn start:dev
```

7. Execute the SEED

```
http://localhost:3000/api/seed
```

8. All the API's documentation will be available at the endpoint

```
http://localhost:3000/api
```
